# Haskell Spotify TUI <br/> FFP-Project 

## Run

```sh
stack run
```

## Build and Run Executable

```sh
stack build
.stack-work/dist/*/*/build/haskell-spotify-tui/haskell-spotify-tui
```

## Install and Run on System

```sh
stack install
haskell-spotify-tui
```
